KronIDE

KronIDE is a lightweight, self-hosted web IDE, designed specifically for PHP-based environments and “place-and-run” deployments. No Node.js, no build tools — just pure PHP, jQuery and Bootstrap.

View Features

Features

  • Fast and simple web-based code editor
  • Secure file browser with workspace isolation
  • Built-in terminal (restricted mode)
  • Syntax highlighting with Ace.js
  • Dark UI inspired by VSCode
  • Full PHP 7.4 compatibility
  • No dependencies outside PHP, jQuery, Bootstrap
